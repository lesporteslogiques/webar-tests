# Découverte de A-frame et de la réalité augmentée avec AR.js

Ce dépôt est un regroupement de quelques fichiers html qui ont servi a découvrir et expérimenter un peu le framework A-frame et AR.js. L'objectif était de pouvoir interagir avec un modèle affiché sur un marqueur en réalité augmentée en cliquant dessus (ou en le touchant sur un téléphone). Si ceci ne s'est finalement pas fait, il n'empêche que l'affichage et l'animation de modèles 3D dans une page web reste intéressante et offre beaucoup de possibilités.

# Préambule
Bibliothèques utilisées
  * a-frame : https://aframe.io/
  * AR.js : https://ar-js-org.github.io/AR.js-Docs/

## 1-embedded-test
Le premier test, avec une scène 3D intégrée dans une page HTML basique. Ici, je me suis familiarisé avec les bases de A-frame.

## 2-ps2-spheres
Afficher des modèles, c'est chouette. Mais les animer, c'est mieux ! Plusieurs sphères tournent en orbite autour d'un même point. L'idée était de (tenter de) reproduire l'animation des boules lumineuses du menu de la PlayStation 2.

## 3-realite-augmentee
Le premier test de réalité augmentée. L'objectif était de se familiariser avec l'AR.

## 4-ar-animation
Qu'est-ce qui est mieux qu'un modèle en AR ? Un modèle animé en AR ! Ici, l'objectif était simplement de faire pivoter un objet autour d'un axe. Deux méthodes différentes sont utilisées ici, plus d'informations dans le code en commentaire.
